# daw-php

# How to use Docker

Run the Docker container (one time) :

`docker-compose up -d`

Distroy the Docker container :

`docker-compose down`

Stand-by the Docker container :

`docker-compose stop`

Wake-up the Docker container :

`docker-compose start`

Login into Docker container :

`docker-compose exec daw-php bash`
